* Teal Swan Frequency Painting Downloader
This program downloads paintings from Teal Swan's website.

https://tealswan.com/paintings/

** To run:
Download the latest release, `tsdl.jar`.

Then run the following:
#+begin_src shell
java -jar tsdl.jar
#+end_src
